# Grade Average Calculator

## Overview

Grade Average Calculator is a C# console application developed to calculate course averages, letter grades, GNO, and GPA. It also allows users to manage course information and view statistics.

## Features

* Add new courses
* Remove existing courses
* Display all courses
* Clear all course records
* Calculate course average (40% Midterm + 60% Final)
* Calculate letter grades using two different grading systems:

  * GNO (AA, BA, BB, ...)
  * GPA (A, A-, B+, ...)
* Determine pass/fail status
* Calculate GNO
* Calculate GPA
* Display statistics:

  * Total number of courses
  * Passed and failed courses
  * Success rate
  * Average score
  * Total credits
  * Highest grade course
  * Lowest grade course

## Technologies Used

* C#
* .NET
* Console Application
* List
* Dictionary
* Methods
* Loops (for, foreach, while, do-while)
* Conditional Statements
* Exception-safe input validation using `TryParse`

## How to Run

1. Open the project in Visual Studio.
2. Build and run the application.
3. Use the menu to manage courses and calculate grades.
